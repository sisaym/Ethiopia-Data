Developers can use the API in the page to query the data and get outputs in the form of XML and JSON.

Please note that currently only read is allowed through the API and other CRUD operations are disabled, see the details below on how to use the API for each of the data sets.

Introduction

The api page for the data sets is available at data/api/resource_name, where resource_name is the resource name for the data sets

    The resource name for the data sets available is
  • Crop Statistics: crop_data
  • Import statistics: imports
  • Export statistics: exports

The following brief example shows how to retrieve the data using the resource names for agricultural statistics

Going to api/data/crop_data will give xml data for crop statistics.

click here to give it a try

Agricultural statistics

The crop statistics API allows to retrieve data related to crop production, area coverage, yield and number of farmers growing a crop.

In addition the data api allows the data to be filtered by year, and level of data.

The resource_name for agricultural/crop statistics is crop_data, going to api/data/crop_data will give data for crop statistics.

To get xml data with 100 records write api/data/crop_data/?format=xml&limit=100

click here to give it a try

To get json data with 100 records write api/data/crop_data/?format=json&limit=100

click here to give it a try

Import statistics

Import api allows to retrieve import statistics for Ethiopia by item, year, and origin.

The resource_name for import statistics is imports, going to api/data/imports will give data for import statistics.

To get xml data with 100 records write api/data/imports/?format=xml&limit=100

click here to give it a try

To get json data with 100 records write api/data/imports/?format=json&limit=100

click here to give it a try

Export statistics

Import api allows to retrieve export statistics for Ethiopia by item, year, and destination.

The resource_name for import statistics is exports, going to api/data/exports will give data for exports statistics.

To get xml data with 100 records write api/data/exports/?format=xml&limit=100

click here to give it a try

To get json data with 100 records api/data/exports/?format=json&limit=100
